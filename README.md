# react-native-raw-pointer

Generic raw pointer/touch event listener for React Native

## Installation


```sh
npm install react-native-raw-pointer
```


## Usage


```js
import { RawPointerView } from "react-native-raw-pointer";

// ...

<RawPointerView color="tomato" />
```
